Pass by Value vs Reference

What does it mean when we pass value by reference? Note: Function arguments are always passed by value and are copied into the function.
Primitive types
let a = "KYH"
let b = a
a = "Hello" // We assign a new value to a
console.log(b) // We retain the copied value

Objects
let objA = { name: "Kal", lastname: "Doe" }
let objB = objA
objA.name = "Ada" // we change the value of prop name
console.log(objB) // prop name has changed
objB here points to the location of the original object. It's not a copy of objA.
Note: You can define how/if a prop can be accessed/changed by using the corresponding Object constructor method.
See more about define access to props; https://stackoverflow.com/questions/7757337/defining-read-only-properties-in-javascript See more about objects passed by reference; https://stackoverflow.com/questions/16880418/javascript-pas
