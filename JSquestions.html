Primitive Data Type

What are primitive datatypes? Understanding the above lets proceed Primitive types are immutable! But what does it really mean...?
let foo = "Hello World" console.log(foo.toUpperCase()) // we are not mutating foo foo.toUpperCase() // we use a method on foo...
console.log(foo) // but the original value is unchanged
We can do this, console.log(foo[0]) // "H" But not this(immutable), foo[0] = "T" // "Hello World" Assigning a value gives it a new value it does not mutate the original value as the case with objects Noteworthy; in React 'props'(the object) can not be mutated, by design...
See more; https://developer.mozilla.org/en-US/docs/Glossary/Primitive More about objects passed by reference not by copy; https://stackoverflow.com/questions/16880418/javascript-pas
